Lars T Hansen has a program called ffigen, which takes C header files
and turns them into S-expressions:

  <http://www.ccs.neu.edu/home/lth/ffigen/>

It's designed to be the frontend to a ffi generator, and uses the lcc
compiler and can handle all of ANSI C. It could be useful for goo.
