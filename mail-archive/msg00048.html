We all know that real programmers use Emacs (ducks).  However,
some of the world have become addicted to user-friendly integrated
development environments, like Microsoft's Visual Studio platform,
or Borlands Builder tools.

Now Goo programmers can share in the excitement, using a new
plugin for the Eclipse platform.

The Eclipse project is an Open Source editing platform released
by IBM (and is apparently the basis for their Websphere product
line.)  It offers an immense number of tools and widgets useful
for building graphical and text-based editing tools (among other
things.)

I was curious how difficult it would be to create an Emacs-style
editing mode for Goo using this framework, so I build the
imaginatively named "Goo Ide", which supports a small set of the
many features available to Java programmes under Eclipse.

I have written up a short document describing my impressions of
this work at
http://people.debian.org/~bfulgham/goo/goo_writeup/Goo_Document.html.

You can download the Goo Ide plugin from the same place as my Debian
packages:
http://people.debian.org/~bfulgham/goo/

Have fun with Goo! (and Eclipse!)

-Brent
