v0_139 available

  • Subject: v0_139 available
  • From: Jonathan Bachrach <jrb@xxxxxxxxxx>
  • Date: 19 Jul 2002 11:54:47 -0400
we're happy to announce a new version of goo, v0_139, that improves
performance, supports a first crack at threads on linux, includes a
bunch of patches, and includes a new installation mechanism.  the new
installation tarball is platform independent and comes in the devo
form only for now.  consult the doc for installation procedures.  we
will make a windows binary release available soon.  we have yet to
confirm macosx compatibility but should have something by next week.
this is a major new release and we would appreciate any feedback.

we will working to improve the pthreads-based threads implementation
and will start looking to porting it to windows.  any help on this
would be appreciated.  the major obstacle might be that the boehm gc
supports threads but only when compiled with vc++.

upcoming releases will include networking support and the beginnings
of incremental whole program analysis.  please send in any other
requests.

jonathan bachrach
