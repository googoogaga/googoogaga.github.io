Microsoft Visual C++ 6. The only change required is to add goo/x.c to the

makefile to something that works with MSVC. I've uploaded a distribution
that includes the changed build.bat and a binary built using MSVC to:

  http://www.double.co.nz/goo/vcgoo154a.zip

OpenGL program also works as long as you remove the call to 'usleep'
which MSC doesn't have.
