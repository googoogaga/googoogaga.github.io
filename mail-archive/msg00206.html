I've updated Goo.NET to work with release v146 now. This includes
support for calling .NET framework classes and methods. Examples are
shown of retrieving a web page and displaying a message box. Threads
also work with this version which makes it much more useful than my
previous .NET release.

For details and download see:

http://radio.weblogs.com/0102385/stories/2002/05/04/goodotnet.html

The archive also contains the update %net.c code for windows and a
normal windows .exe binary.
